# bilibiliUnfollowinghelper
B 站批量取关助手 | 简洁安全的 B 站关注管理工具
一款基于 Python + PyQt5 开发的B 站批量取关工具，专注解决 B 站官方无法批量管理关注列表的痛点，界面简洁、操作简单、本地运行更安全，支持一键筛选、批量取消关注，让关注管理更高效。
核心功能
批量取关：支持全选 / 多选关注 UP 主，一键批量取消关注，告别手动逐个点击
分页加载：自动加载全部关注列表，无数量限制，清晰展示用户信息
可视化界面：轻量化 GUI 窗口，操作直观，无需代码基础，开箱即用
配置持久化：支持保存登录状态、操作配置，重启软件无需重复设置
工具亮点
安全优先：不窃取账号信息，不采集用户数据，本地离线管理关注列表
简单易用：扫码登录 / Cookie 登录双模式，一键登录、一键操作
轻量无捆绑：纯绿色工具，无广告、无插件、无后台驻留
适配友好：Windows 全平台兼容，运行流畅，资源占用低
适用场景
关注人数过多，想要快速清理关注列表的用户
厌倦手动逐个取关，追求高效管理的 B 站用户
需要批量整理关注、优化账号状态的用户
安全声明
本工具仅用于本地账号管理，调用 B 站官方开放接口 不破解、不篡改、不滥用接口，无任何恶意代码
所有登录凭证仅保存在用户本地，杜绝信息泄露风险
请合理使用工具，避免高频操作导致账号风控
有很多bug,不要告诉我,大部分是ai实现的(接口是抄别人的方法)
